## Local Instance

1. `git clone https://github.com/kdreagle/lightspeed.git`
2. `cd lightspeed`
3. `npm install`
4. `SET DB_PASS=Password123&& yarn dev`

## Stack

- Next.js with rbx
- MySQL hosted on Azure
- Hosted on Vercel

## Template Command

`npx create-next-app --example with-rbx-bulma-pro with-rbx-bulma-pro-app`
